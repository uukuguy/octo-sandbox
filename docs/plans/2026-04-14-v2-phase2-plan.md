### S0.T2 — ADR-V2-NEW: Agent loop 多步工作流设计原则

**问题**: D87 暴露 grid-engine agent loop 在 text-only 响应后终止。需要定义**标准化的 agentic loop 行为**，确保三 runtime 一致。

**候选方向**:

| 选项 | 描述 |
|------|------|
| **A. Anthropic SDK 模式** | tool_result → UserMessage → re-invoke LLM → 只在 EndTurn 无 tool 时退出 |
| **B. 显式完成信号** | skill prose 定义结束 marker（如 `<task_complete/>`），loop 在看到 marker 时退出 |
| **C. 最小 tool 执行数** | skill frontmatter 声明 `min_tool_calls`，loop guarantees 执行 |
| **D. 混合** | A 为主（默认），B/C 作为 opt-in |

**建议**: **D（Anthropic 模式为主 + 显式完成信号作为 skill 强约束）**

**前置**: 无，可与 S0.T1 并行
