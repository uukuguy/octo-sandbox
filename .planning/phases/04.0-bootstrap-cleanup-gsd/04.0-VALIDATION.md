## Phase Gate (Composite Check)

After all 5 tasks complete, run this single composite to confirm phase ready for `/gsd-verify-work` / `/gsd-end-phase`:

```bash
bash -c '
  set -e
  echo "Phase 4.0 gate check:"

  # CLEANUP-01: chunk_type sweep + provenance marker
  grep -F "<!-- @chunk-type-sync ADR-V2-021 -->" docs/design/EAASP/L1_RUNTIME_ADAPTATION_GUIDE.md > /dev/null && echo "✓ CLEANUP-01 marker"
  ! grep -nE "\| *\"(text|tool_call|hook_fired|pre_compact)\" *\|" docs/design/EAASP/L1_RUNTIME_ADAPTATION_GUIDE.md > /dev/null && echo "✓ CLEANUP-01 stale strings removed"

  # CLEANUP-02: D120 single-line grep returns closed
  first_d120=$(grep "D120" docs/design/EAASP/DEFERRED_LEDGER.md | head -1)
  echo "$first_d120" | grep -q "closed" && echo "✓ CLEANUP-02 D120 first-line closed"
  grep -qE "row-edit-on-close|关闭/状态变更" docs/design/EAASP/DEFERRED_LEDGER.md && echo "✓ CLEANUP-02 preamble convention"

  # CLEANUP-03: strategy checklist exists with 5+ items
  test -f docs/reviews/strategy-grid-two-leg-checklist.md && echo "✓ CLEANUP-03 file"
  [ $(grep -c "^- \[ \]" docs/reviews/strategy-grid-two-leg-checklist.md) -ge 5 ] && echo "✓ CLEANUP-03 ≥5 items"

  # CLEANUP-04: CODEOWNERS with 5 paths assigned to @firmwwwee
  test -f .github/CODEOWNERS && echo "✓ CLEANUP-04 file"
  [ $(grep -cE "@firmwwwee" .github/CODEOWNERS) -ge 5 ] && echo "✓ CLEANUP-04 ≥5 reviewer entries"
  grep -qE "^crates/grid-server/" .github/CODEOWNERS && echo "✓ CLEANUP-04 filesystem-correct path"

  # GOVERNANCE-01: PLAN.md has ≥4 review_protocol annotations
  [ $(grep -c "review_protocol:" .planning/phases/04.0-bootstrap-cleanup-gsd/*-PLAN.md) -ge 4 ] && echo "✓ GOVERNANCE-01 review_protocol present"

  echo "── Phase 4.0 gate PASS ──"
'
```
